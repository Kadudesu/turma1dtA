    LOOP WHILE

    O processo inicia quando um laço while é encontrado durante a execução de um programa. A condição é avaliada para ver se ela é a verdadeira (True) ou falsa (False).
    Se a condição for True, as instruções que pertencem ao laço serão executadas.
    A condição laço While é verificada novamente. Se a condição for True novamente, a sequência de instruções é executada novamente e o processo se repete.
    Quando a condição for False, o laço é interrompido (Loop Stops) e o programa continua para além do laço

    LOOP FOR

    Um loop for é usado para iterar sobre uma sequência.

    Isso é menos parecido com a palavra-chave for em outras linguagens de programação e funciona mais como um método iterador encontrado em outra linguagens de programação orientada a objeto.

    Com o Loop podemos executar um conjunto de intruções, uma vez para cada item em uma lista.
